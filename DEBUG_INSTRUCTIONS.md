# 🔍 4月シフト表示問題のデバッグ手順

## 1. ブラウザのコンソールログ確認

1. **F12**キーまたは**右クリック → 検証**でデベロッパーツールを開く
2. **Console**タブを選択
3. 以下のログが表示されているか確認：

### 期待されるログ：
```
📦 データサービス: Supabase
✅ Supabaseモードで動作中
🔄 2026年4月のシフトを購読開始
🔄 Supabaseサブスクリプション開始: 2026年4月
  購読期間: 2026-04-01 〜 2026-04-30
📅 2026年4月のシフトを読み込み中...
  期間: 2026-04-01 〜 2026-04-30
  取得したシフト数: 5件
  初回読み込み: 5件のシフト
📊 2026年4月のシフトを受信: 5件
```

## 2. ページを強制リロード

- **Ctrl + Shift + R**（Windows）または **Cmd + Shift + R**（Mac）でキャッシュを無視してリロード

## 3. コンソールで直接確認

デベロッパーツールのConsoleで以下のコマンドを実行：

```javascript
// 現在の年月を確認
console.log('現在の年月:', window.location.href);

// Supabaseモードか確認
console.log('USE_SUPABASE:', import.meta.env.VITE_USE_SUPABASE);
```

## 4. エラーメッセージの確認

コンソールに赤文字のエラーメッセージがないか確認してください。
