# シフト表 操作マニュアル

このドキュメントは、シフト表画面でできる操作をまとめたものです。
新しく入った職員さんがこれを読めばひと通りシフトを組めるようになることを目指しています。

---

## 1. 画面の見方

シフト表は **横軸＝日付**、**縦軸＝ヘルパー** のマス目（セル）で構成されます。
1つのセルは **4行構造** になっており、上から順に次の意味を持ちます。

| 行 | 内容 | 例 |
|----|------|-----|
| 1行目 | 開始時間〜終了時間 | `9:00-10:30` |
| 2行目 | 利用者名（サービス種別） | `田中様(身体)` |
| 3行目 | 稼働時間（自動計算） | `1.5` |
| 4行目 | メモ・備考 | 自由記入 |

時間を入れると 3行目の稼働時間は自動で計算されます。

### サービス種別と背景色

セルの色でサービス種類が一目で分かるようになっています。

| 種別 | 色 | 種別 | 色 |
|------|----|----|----|
| 身体 | 黄色 | 同行 | 明るい緑 |
| 家事 | 薄オレンジ | 深夜 | 濃い青 |
| 重度 | オレンジ赤 | 深夜(同行) | 濃い紫 |
| 行動 | グレー | 通院 | 青 |
| 移動 | 緑 | 休み希望 | ピンク |
| 事務 | インディゴ | 指定休 | 緑 |
| 営業 | バイオレット | 予定 | 紫 |
| 会議 | 紫 | キャンセル | 赤 |

---

## 2. シフトを記入する

### 2-1. クリックして書く

1. 入れたいセルを **1回クリック** すると青枠で選択されます。
2. そのまま文字入力（数字キーや時間）を打つと、**自動的に編集モード** に切り替わります。
3. **F2キー** または **ダブルクリック** でも編集モードに入れます。

### 2-2. 入力ルール

- 1行目に時間を `9:00-10:30` のように入力 → 3行目に `1.5`（時間）が自動で入ります。
- 2行目をクリックすると **利用者・ケア内容・サービス種別** を選ぶモーダルが開きます。
- サービス種別を選ぶと、セルの背景色が自動で切り替わります。

### 2-3. 移動キー

| キー | 動き |
|------|------|
| `Enter` | 下のセルへ移動 |
| `Shift + Enter` | 上のセルへ移動 |
| `← ↑ → ↓` | 矢印方向のセルへ移動 |
| `Esc` | 編集を中止／選択解除 |
| `Backspace` / `Delete` | セル内容を消す |

---

## 3. ケア内容を変える・選び直す

2行目（利用者名と種別が表示されている行）をクリックすると、選択モーダルが開きます。
- 利用者を選び直す
- サービス種別（身体／家事／通院 など）を選び直す
- 既存の入力に上書きする

サービス種別を変えれば、セルの色も自動で変わります。

---

## 4. シフトを移動する（ドラッグ＆ドロップ）

別の日や別のヘルパーへシフトを **そのまま動かしたい** ときに使います。

1. 動かしたいセルを **長押し → ドラッグ** します（半透明になります）。
2. 動かしたい先のセルへ **ドロップ** します。
3. 移動先にすでにシフトがある場合は **入れ替え（スワップ）** されます。
4. 表の端までドラッグすると **自動でスクロール** します。

> 「移動先のシフトが消えた！」と思ったら、ほとんどの場合スワップで元の場所に移っています。Ctrl+Z で元に戻せます。

---

## 5. シフトをコピー＆ペーストする

### 5-1. 1セルだけコピー

| 操作 | キー |
|------|------|
| コピー | `Ctrl + C` （Macは `Cmd + C`） |
| 貼り付け | `Ctrl + V` （Macは `Cmd + V`） |

### 5-2. 複数行まとめてコピー

複数のセルを選択してから `Ctrl + C` すると、まとめて記憶されます。
貼り付け先で `Ctrl + V` すると、行をずらしながら順番に貼り付きます。

### 5-3. Excel／スプレッドシートからの貼り付け

タブ区切りの表データをそのままペーストすると、表形式のまま貼り付きます。
> **注意：キャンセル状態はコピーされません。** 貼り付け先は通常のシフトとして入ります。

---

## 6. キャンセル処理（時間あり／時間なし）

利用者からキャンセルの連絡が来たときは、セルを **右クリック** してキャンセル処理を行います。
このアプリは **2種類のキャンセル** を区別します。

### 6-1. キャンセル（時間を残す）

- 用途：**直前キャンセル料が発生する**ケース、ヘルパーの給与計上が必要なケース
- セルが **赤色** に変わります
- 3行目の稼働時間は **そのまま残る** ので、給与計算に反映されます

### 6-2. キャンセル（時間を消す）

- 用途：**料金も給与も発生しない**完全キャンセル
- セルが **赤色** に変わります
- 3行目の稼働時間は **0** になり、給与には計上されません

### 6-3. キャンセルを取り消す

キャンセル済みのセルを右クリックして「キャンセル取消」を選ぶと、
元のサービス種別の色に戻り、稼働時間も復活します。

### 6-4. 一括キャンセル

複数行を選択した状態で右クリックすれば、選択した全シフトを **まとめてキャンセル** できます。

---

## 7. 削除する／取り消す

### 7-1. 削除

- セルを右クリック → 「ケア削除」
- 複数行選択中なら、右クリックメニューに「**N件選択中**」と表示され、まとめて削除できます。

### 7-2. やり直し（Undo / Redo）

| 操作 | キー |
|------|------|
| 元に戻す | `Ctrl + Z` （Macは `Cmd + Z`） |
| やり直し | `Ctrl + Shift + Z` （Macは `Cmd + Shift + Z`） |

セルの背景色も含めて元の状態に戻ります。
**間違えても慌てずに `Ctrl + Z`** が鉄則です。

---

## 8. 範囲選択・一括操作

| 操作 | やり方 |
|------|--------|
| 連続範囲選択 | `Shift + クリック` で範囲指定 |
| 飛び飛びで選択 | `Ctrl + クリック`（Macは `Cmd + クリック`） |
| 選択解除 | `Esc` |

選択した状態で右クリックすると、

- 一括削除
- 一括キャンセル（時間残す／時間消す）
- 一括コピー

ができます。

## 10. よくあるトラブル

| 症状 | 対処 |
|------|------|
| セルが動かない／入力できない | 一度 `Esc` を押してから、セルを1回クリックし直す |
| ドラッグでスワップしてしまった | `Ctrl + Z` で元に戻す |
| キャンセル後に給与が変わらない | 「時間残す」キャンセルを使っているか確認（時間ありは給与に反映） |
| 色がおかしい | 2行目を開き直してサービス種別を選び直す |
| ペーストが効かない | コピー元が **キャンセル状態だと貼り付かない仕様**。元のシフトをコピーし直す |
