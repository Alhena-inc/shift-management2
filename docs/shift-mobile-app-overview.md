# shift-mobile-app 概要

## 概要

shift-mobile-app は訪問介護事業所のスタッフ向けモバイルWebアプリ（PWA対応）。スタッフは自分のシフトを確認し、希望出勤（勤務可能日時）を提出できる。

本番URL: `https://shift-management2.vercel.app/`

## アーキテクチャ

| 項目 | 内容 |
|------|------|
| Frontend | React + TypeScript + Vite |
| Styling | Tailwind CSS |
| Database | Supabase (Postgres) / Firebase Firestore |
| Auth | Google OAuth (招待制) |
| PWA | manifest.json + Service Worker |

## シフト関連の主要画面

### 1. ログイン画面 (`/`)
- Google OAuth ログインボタン
- 事前登録メールアドレスのみ許可
- 招待制の旨を記載

### 2. シフト一覧（個人シフト表）(`/personal/:token`)
- ヘルパー個人のトークン付きURL
- 当月のシフト一覧をカレンダー形式で表示
- 月切り替え（前月/翌月）ボタン
- 各シフトはサービス種別（身体介護・生活援助等）で色分け
- リアルタイム更新（Firestore/Supabase Realtime）

### 3. シフト詳細
- 個別シフトをタップで詳細表示
- 利用者名・サービス種別・時間帯・場所
- 備考欄

### 4. 希望出勤（勤務可能）入力
- 月〜日の希望時間帯を入力
- 備考欄に「祝日NG」等を記入
- 提出ボタンで送信
- 既存の希望はピンク背景でセル表示

### 5. 通知設定
- PWA インストール案内
- 通知の有効/無効
- ホーム画面追加ガイド

## 操作フロー（スタッフ視点）

1. ログイン画面から Google アカウントでログイン
2. シフト一覧画面が表示される（自動的に当月）
3. 月切り替えで前月/翌月を確認
4. 気になるシフトをタップして詳細確認
5. 希望出勤画面へ移動
6. 月〜日の希望時間帯を入力
7. 備考欄に注意事項を記入
8. 提出ボタンで送信
9. 通知設定でPWAをホーム画面に追加

## モバイルUI設計のポイント

- 縦長レイアウト（390x844想定）
- 大きめタップ領域（44px以上）
- 重要操作はカラーアクセント（青系）
- カレンダーは横スクロール対応
- フッターナビゲーション（シフト/希望/通知）
