# シフト操作 動画撮影フロー（録画台本）

このドキュメントは、シフト表の操作説明動画を撮影するための台本です。
シーンごとに **「映すもの」「操作」「ナレーション例」「撮影のコツ」** を整理しています。

撮影前チェックリスト
- [ ] 個人情報（実在の利用者名・電話番号など）が映らないようテスト用データに切り替えたか
- [ ] ブラウザのブックマークバー・通知ポップアップを非表示にしたか
- [ ] マウスカーソル拡大／クリックエフェクトをONにしたか（操作が見やすくなる）
- [ ] 画面解像度は 1920×1080 推奨
- [ ] 1シーンごとに録画を区切ると編集が楽

---

## シーン 1：オープニング・画面の見方

**映すもの**：シフト表全体

**操作**
1. シフト表画面を開く
2. 1つのセルをゆっくりズーム（マウスをホバーして指し示す）
3. 4行構造を上から順に指す

**ナレーション例**
> 「シフト管理アプリの操作方法を説明します。まず画面の見方から。
> 横が日付、縦がヘルパーさん。1つのマスは4行に分かれていて、
> 上から **時間 → 利用者名とケア内容 → 稼働時間 → メモ** の順に並びます」

> 「色でケア内容が分かるようになっていて、たとえば黄色は身体、薄オレンジは家事、青は通院、赤はキャンセル、というふうに色分けされています」

**撮影のコツ**
- 色の説明は別撮りで「色見本一覧」を画面に出すか、Notion／PDFを別ウィンドウで見せると分かりやすい
- マウスホバーで色のついたセルを順番に指していくとリズムが良い

## シーン 2：シフトを記入する（基本）

**映すもの**：空のセル

**操作**
1. 空のセルを **1回クリック** → 青枠が出るのを見せる
2. そのままキーボードで `9:00-10:30` と打つ
3. 1行目に時間が入り、3行目に `1.5` が自動表示されるのを見せる
4. `Esc` で確定

**ナレーション例**
> 「シフトを入れたいマスを1回クリックします。青い枠が出たら選択中のサインです」

> 「そのまま時間を打ってみますね。`9時から10時半` と入力すると…
> 3行目に自動で `1.5時間` と稼働時間が入りました。これは自分で計算しなくて大丈夫です」

> 「F2キーやダブルクリックでも編集モードに入れます」

**撮影のコツ**
- キー入力をオンスクリーンで表示するツール（`KeyCastr` など）を使うと「何を押したか」が伝わる
- 自動計算が出る瞬間は **0.5秒ほど止める** と「おっ」と分かる

## シーン 3：ケア内容を変える

**映すもの**：時間だけ入っているセル

**操作**
1. **2行目をクリック** → モーダル（選択画面）が開く
2. 利用者を選ぶ
3. ケア内容（身体・家事・通院など）を選ぶ
4. 確定 → セルの色が変わる様子を見せる

**ナレーション例**
> 「次に、誰のところに行くか・何をするかを設定します。
> 2行目をクリックすると、利用者さんとケア内容を選ぶ画面が開きます」

> 「利用者さんを選んで、ケア内容を選びます。今回は身体介護にしますね。
> ほら、マスの色が黄色に変わりました。色でひと目で分かるようになっています」

**撮影のコツ**
- 種別を **2〜3パターン** 切り替えて色が変わる様子を見せると効果的
- 「身体→通院→家事」と切り替えるとカラフルで印象に残る

## シーン 4：時間入力と自動計算（応用）

**映すもの**：別のセル

**操作**
1. `13:30-15:00` と入力 → 1.5
2. `9:00-12:00` と入力 → 3.0
3. `Enter` で下のセルに移る
4. 矢印キーで移動も見せる

**ナレーション例**
> 「時間の入れ方をいくつか試してみますね。
> `13:30-15:00` で1.5時間、`9:00-12:00` で3時間、と自動で計算されます」

> 「`Enter` を押すと下のセルへ、矢印キーで上下左右に動けます。
> マウスを使わなくても連続で入力できるので慣れると速いです」

**撮影のコツ**
- キーボードショートカットを字幕で出す（`Enter ↓`、`Shift+Enter ↑` など）

---

## シーン 5：シフトを移動する（ドラッグ＆ドロップ）

**映すもの**：すでに入力済みのセル

**操作**
1. 入力済みセルを **長押し** → ドラッグ開始（半透明になる）
2. 別の日のセルへドロップ → 移動完了
3. 別パターン：すでに別シフトが入っているマスへドロップ → **スワップ（入れ替え）** されるのを見せる
4. `Ctrl+Z` で戻す

**ナレーション例**
> 「シフトを別の日や別のヘルパーに動かしたいときは、ドラッグ＆ドロップが使えます。
> マスを長押ししてズルッと引っ張って、移したい場所で離します」

> 「もし移動先にすでにシフトがあった場合は、**入れ替え** になります。
> 『あれ、消えた？』と思ったら、ほとんどの場合は元のマスに移っているだけです。
> もし違ったら `Ctrl+Z` で戻せるので安心してください」

**撮影のコツ**
- スワップの様子は **ビフォー／アフターの静止画** を撮っておくと編集で分かりやすい
- ドラッグ中の半透明状態を **ゆっくり** 見せる

## シーン 6：コピー＆ペースト

**映すもの**：入力済みセル

**操作（3パターン）**

**6-A：1セルだけコピー**
1. セルを選択 → `Ctrl+C`
2. 別のセルへ移動 → `Ctrl+V`

**6-B：複数行まとめてコピー**
1. `Shift+クリック` で複数セルを範囲選択
2. `Ctrl+C`
3. 別の日へ移動 → `Ctrl+V` で連続貼り付け

**6-C：Excelから貼り付け**
1. Excel（またはGoogleスプレッドシート）で表をコピー
2. シフト表のセルをクリック → `Ctrl+V`
3. 表形式のまま入る様子を見せる

**ナレーション例**
> 「同じシフトを別の日にも入れたいときは、コピー＆ペーストが便利です。
> マスを選んで `Ctrl+C`、貼り付けたい場所で `Ctrl+V`」

> 「複数のマスをまとめてコピーすることもできます。
> `Shift` を押しながらクリックして範囲を選んで、`Ctrl+C` でまとめてコピー」

> 「Excelやスプレッドシートからもそのまま貼り付けられます。表のレイアウトはそのまま入ります」

> 「ひとつ注意点。**キャンセルになっているシフトはコピーしても通常状態で貼り付きます**。
> キャンセルのまま複製したい場合は、貼り付けた後にもう一度キャンセル処理をしてください」

**撮影のコツ**
- 3パターンそれぞれで小見出し（テロップ）を入れると親切
- Excelからの貼り付けはサンプル用Excelファイルを事前に用意

## シーン 7：キャンセル処理（時間あり／時間なし）

**ここが一番大事なシーン。ゆっくり丁寧に。**

**映すもの**：入力済みセル（できれば3〜4個並んでいる状態）

**操作**

**7-A：キャンセル（時間を残す）**
1. セルを **右クリック** → メニュー表示
2. 「キャンセル（時間残す）」を選ぶ
3. セルが赤くなる
4. 3行目の **稼働時間がそのまま残っている** のをアップで見せる

**7-B：キャンセル（時間を消す）**
1. 別のセルを右クリック → 「キャンセル（時間削除）」
2. セルが赤くなる
3. 3行目が **0になっている** のをアップで見せる

**7-C：キャンセル取り消し**
1. 赤くなっているセルを右クリック → 「キャンセル取消」
2. 元の色に戻る

**ナレーション例**
> 「キャンセルの連絡が来たときの操作です。
> このアプリには **2種類のキャンセル** があります。間違えると給与計算に影響するので、ここはしっかり覚えてください」

> 「**1つめは『時間を残すキャンセル』**。直前キャンセル料が発生するときや、ヘルパーさんに給与を計上したい場合に使います。
> 右クリックして『キャンセル（時間残す）』を選ぶと、マスが赤くなりますが、稼働時間はそのまま残ります。給与計算に反映されます」

> 「**2つめは『時間を消すキャンセル』**。料金も給与も発生しない、完全なキャンセルのときに使います。
> 同じく赤くなりますが、こちらは稼働時間が0になります。給与には計上されません」

> 「間違って押してしまっても、もう一度右クリックして『キャンセル取消』を選べば元に戻せます」

**撮影のコツ**
- **3行目の稼働時間をズームアップ** で見せるのが超重要
  - 時間残す：`1.5` のまま
  - 時間消す：`0`
- 字幕で「給与に反映される／されない」を強調
- 同じセルで両パターンをやると比較しやすい

## シーン 8：削除・元に戻す

**操作**
1. セルを右クリック → 「ケア削除」
2. セルが空になる
3. `Ctrl+Z` → 元に戻る
4. `Ctrl+Shift+Z` → またやり直し

**ナレーション例**
> 「シフトをまるごと消したいときは、右クリックから『ケア削除』。
> 間違えても **`Ctrl+Z`** で元に戻せます。やり直したいときは **`Ctrl+Shift+Z`**」

> 「**間違えても慌てずに `Ctrl+Z`**、これだけ覚えておけば大丈夫です」

**撮影のコツ**
- 「Ctrl+Z」は2〜3回連続で押して **何ステップでも戻せる** ことを見せる

## シーン 9：範囲選択・一括操作

**操作**

**9-A：連続範囲選択**
1. 1つめのセルをクリック
2. `Shift+クリック` で別のセル → 間が全部選択される

**9-B：飛び飛び選択**
1. 1つめをクリック
2. `Ctrl+クリック`（Macは`Cmd+クリック`）で別のセルを追加選択

**9-C：一括キャンセル**
1. 範囲選択した状態で右クリック
2. メニューに「**N件選択中**」と表示される
3. 「キャンセル（時間残す）」を選ぶ → 全部赤になる

**ナレーション例**
> 「複数のシフトを一気に操作したいときは、まとめて選択できます」

> 「`Shift+クリック` で連続範囲、`Ctrl+クリック` で飛び飛びの選択ができます」

> 「選択した状態で右クリックすると、『何件選択中』と表示されて、まとめてキャンセルしたり、まとめて削除したりできます。
> インフルエンザで何日か連続キャンセル、なんてときに便利です」

**撮影のコツ**
- 「N件選択中」のテロップが出る瞬間を **アップで** 見せる
- 一括キャンセルの実例（実際の業務シーン）でナレーションすると説得力UP
