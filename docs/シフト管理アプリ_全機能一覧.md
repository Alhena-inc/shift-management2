# シフト管理アプリ 全機能一覧

訪問介護事業所向けシフト管理アプリの全機能を網羅したリファレンスです。
各機能は「**何ができるか**」「**誰が使えるか**」「**入力／出力**」の観点で整理しています。

権限区分：
- **管理者（admin）**：全機能利用可
- **スタッフ（staff）**：シフト表・給与（自分の分）・休み希望・ケア日誌のみ

## 1. ホーム画面

**用途**：アプリの入口。各機能へのナビゲーションと当日の業務サマリ。

**できること**
- 当日のケア日誌記入状況をダッシュボード表示（管理者のみ）
- 当日のシフト件数の確認
- 各機能カード（シフト管理／ヘルパー管理／利用者管理／給与明細／シフト一括追加／ケア日誌／運営指導書類）からの遷移
- ログイン中のユーザー名・権限バッジ表示
- 権限管理画面への遷移（管理者のみ・歯車アイコン）

**特記**：権限に応じてメニューカードが自動的にフィルタされます。

## 2. シフト管理

> シフト管理アプリの中心機能。詳細は `docs/シフト操作マニュアル.md` 参照。

### 2-1. セル入力・編集

- セル4行構造：時間／利用者名(種別)／稼働時間（自動計算）／メモ
- クリック → そのまま入力で編集モード
- F2キー／ダブルクリックで編集モード
- 矢印キー・Enter・Shift+Enter でセル間移動
- Backspace／Delete でセル内容クリア
- Esc で選択解除

### 2-2. ケア内容（サービス種別）

13種類のサービス種別を色分け表示：

| 種別 | 色 | 時給(参考) |
|------|----|----------|
| 身体 | 黄色 | 2,000円 |
| 家事 | 薄オレンジ | 2,000円 |
| 重度 | オレンジ赤 | 2,000円 |
| 行動 | グレー | 2,000円 |
| 通院 | 青 | 2,000円 |
| 移動 | 緑 | 2,000円 |
| 同行 | 明るい緑 | 1,200円 |
| 深夜 | 濃い青 | 2,000円 |
| 深夜(同行) | 濃い紫 | 1,200円 |
| 事務 | インディゴ | 1,200円 |
| 営業 | バイオレット | 1,200円 |
| 会議 | 紫 | 0円（給与計上なし） |
| 休み希望 | ピンク | — |
| 指定休 | 緑 | — |
| 予定 | 紫 | — |

### 2-3. シフト移動

- ドラッグ＆ドロップで別の日／別ヘルパーへ移動
- 移動先にシフトがあれば **スワップ（入れ替え）**
- 表の端まで引っ張ると自動スクロール

### 2-4. コピー＆ペースト

- `Ctrl+C` / `Ctrl+V`（Macは `Cmd`）
- 複数セル範囲コピー → ペースト先で行ずらし貼り付け
- Excel／スプレッドシートからのタブ区切りペースト対応
- **キャンセル状態は引き継がれない**

### 2-5. キャンセル処理（時間あり／時間なし）

右クリックメニューから2種類のキャンセル：

| 種類 | 給与計上 | 稼働時間 | 用途 |
|------|---------|---------|------|
| キャンセル（時間残す） | ◯ | そのまま | 直前キャンセル料発生時 |
| キャンセル（時間消す） | × | 0 | 完全キャンセル |

「キャンセル取消」で元に戻る。複数選択して **一括キャンセル** 可能。

### 2-6. 削除・元に戻す

- 右クリック →「ケア削除」
- `Ctrl+Z` で元に戻す（複数ステップ可）
- `Ctrl+Shift+Z` でやり直し

### 2-7. 範囲選択・一括操作

- `Shift+クリック`：連続範囲選択
- `Ctrl+クリック`：飛び飛び選択
- 選択後の右クリックで「N件選択中」と表示 → 一括削除／キャンセル／コピー

### 2-8. 月切り替え・リアルタイム同期

- 月切替ボタン（◀ ▶）で前月・翌月へ。年またぎ自動対応
- 月変更時、Supabase から該当月のシフトを再読込
- **複数ユーザーの同時編集をリアルタイム反映**（他端末の編集が即時反映）

### 2-9. 翌月反映

- 「📋 翌月へ反映」ボタン
- 当月の通常シフトを翌月の同じ曜日へ週単位でコピー
- 事務・予定などの事務系サービスは除外

### 2-10. 重複削除

- 「重複削除」ボタン
- 同一日付・ヘルパー・時間帯の重複シフトを検出して削除
- 確認ダイアログで削除件数を表示

### 2-11. 児童名表示

- 児童名が登録されている利用者は、シフト表上で **児童名を主表示** に切り替え

## 3. シフト一括追加

**用途**：テキスト貼り付けで複数シフトを一気に登録。

**できること**
- ヘルパー・年月・デフォルトサービス種別を選択
- テキスト入力例：`2/2 14:00-19:00 三田(身体)住之江区`
  - 日付／時刻／利用者名／(サービス種別)／地区
- 全角→半角の自動正規化、各種ダッシュ統一
- サービス種別17種を自動認識
- 深夜勤務時の日付自動判定（24時超え）
- 解析後プレビューでエラー行を赤くハイライト
- 確認後「シフトに反映」で一括登録

## 4. 休み希望管理

**用途**：ヘルパーの休み希望・指定休を月単位で一括設定。

**できること**
- ヘルパー × 日付の二次元表で管理
- 終日休み or 時間帯指定（5時間スロットから範囲選択）
- ヘルパー単位の全日一括設定／解除
- 日付単位の全ヘルパー一括設定／解除
- 土日祝の自動着色（日曜=赤、土曜=青）
- 表示テキスト自由入力（デフォルト「休」「指定休」）
- 12月→翌1月4日までの月跨ぎ表示
- 一括保存（件数確認表示）

**指定休と休み希望の違い**
- 指定休：常に終日（緑）
- 休み希望：時間帯指定可（ピンク）

## 5. ヘルパー管理

### 5-1. ヘルパー一覧（HelperManagementPage）

- 名前検索
- 雇用形態バッジ表示（パート／正社員／契約／派遣／業務委託）
- 給与情報の一覧表示（時給制：基本時給＋処遇改善／時間、固定給制：基本給＋処遇改善手当）
- カード形式（PC3列／タブレット2列／モバイル1列）
- 「シフト表に入れない」ヘルパーは末尾に表示

### 5-2. ヘルパー詳細（HelperDetailPage）

**基本タブ**
- 氏名（必須）／フリガナ／生年月日／性別
- メールアドレス／電話番号／緊急連絡先
- 郵便番号／住所
- 個人トークン（個人シフト表用URL）
- スプレッドシートID（gid）

**資格タブ**
- 40種類以上の資格から選択追加（看護師・介護福祉士・行動援護従業者など）
- 資格ごとに取得日を記録
- 検索可能なモーダルで複数選択

**給与タブ**
- 時給制：基本時給／処遇改善加算（時間あたり）／オフィス時給
- 固定給制：基本給／処遇改善手当／その他手当（複数）／扶養家族数

### 5-3. ヘルパー操作モーダル（HelperManager）

- 新規ヘルパー追加
- ドラッグ＆ドロップで並び順変更
- 削除（論理削除）
- 個人シフト表URLの生成・コピー・再生成
- シフト表表示名／苗字／名前（経費照合用）の設定
- 給与タイプ（時給／固定給）切替
- 雇用形態の設定
- 「シフト表に入れない」フラグ（給与計算には含めるが表示はしない）

### 5-4. 削除済みヘルパー（admin専用）

- 削除済みヘルパー一覧（PCはテーブル、モバイルはカード）
- 復元機能（ワンクリック）
- 表示項目：名前／メール／権限／削除日時／削除者／削除理由

## 6. 利用者管理

### 6-1. 利用者一覧（CareClientManagementPage）

- 名前・住所・介護度で検索
- 介護度バッジ色分け（要支援1/2、要介護1〜5）
- カード形式（3列グリッド）

**かんたん介護同期機能**
- 事業所名／ユーザーID／パスワードで認証
- 認証情報の保存オプション
- 同期進捗バー、結果サマリ（更新／新規／スキップ／エラー数）
- **毎日深夜2時に自動同期するCronも稼働**

### 6-2. 利用者詳細（CareClientDetailPage）

**基本タブ**
- 氏名・児童氏名・フリガナ
- 性別（本人／児童）
- 生年月日（**和暦年月日ピッカー**：大正〜令和対応）
- 郵便番号／住所／電話／携帯
- 家族の有無
- 契約期間（開始〜終了）／終了理由
- 備考／シフト反映名1・2（重複チェックあり）

**緊急連絡先タブ**
- 連絡先1〜3（氏名／続柄／電話）
- その他連絡先メモ

**利用制度タブ**（有効化したものだけ詳細タブが追加表示）
- 障害者総合支援
- 地域生活支援事業
- 介護保険
- 自費サービス

### 6-3. サービス区分別の管理項目

**介護保険（kaigo）**
- 被保険者証11項目（市町村／要介護度／自己負担限度額など）
- ケアマネジャー／事務所情報
- 訪問介護計画書／通所計画書のアップロード
- 介護支援経過／アセスメント／モニタリング／手順書管理
- 利用サービス組み合わせ（訪問介護＋訪問看護など）

**自費サービス（jihi）**
- サービス提供責任者
- 自費向け訪問介護手順書

**障害福祉（shogai）**
- 支給決定情報（受給額／市町村／支援区分）
- 負担限度額（個別・事業所単位）
- 計画相談支援（初期計画／支援計画）
- 居宅介護計画書（居宅／重度／行動援護／同行援護）
- 提供実績記録
- AI生成対応（Gemini連動）

## 7. 給与計算・給与明細

### 7-1. 給与計算（SalaryCalculation）

- 月別 × 週単位の給与一覧
- サービス種別ごとの時給適用（身体・重度・家事＝2,000円、事務・営業・同行＝1,200円）
- 深夜時間帯（22:00〜翌8:00）は自動で **25%割増**
- 年末年始特別料金（12/31〜1/4）の自動判定
- キャンセル状態（時間残す／時間消す）を反映して集計
- グレーアウトでデータなし週を識別

### 7-2. 給与明細一覧（PayslipListPage、admin）

- 年月別フィルタ
- ヘルパー別の明細表示・編集・削除
- 削除済みヘルパーも該当月にデータがあれば表示
- 複数選択 → **PDF一括出力**
- 出力モード：給与明細／勤怠シート／両方
- 進捗表示
- 未作成のものは「自動生成」ボタンでシフトから一括作成

### 7-3. 時給制給与明細エディタ

- 基本時給＋処遇改善加算
- 日別勤怠（最大31日）：サービス種別ごとの通常時間／深夜時間
- 自動集計：総勤務時間・日数・サービス別
- 深夜割増（×1.25）自動
- 社会保険料：健保／厚生年金／雇用／介護（40歳以上）
- 源泉所得税：年度別税表＋税区分（甲／乙／丙）対応
- 手動調整フラグ（自動計算を無効化して上書き可）
- 年末年始特別手当の加算

### 7-4. 固定給制給与明細エディタ

- 基本給＋処遇改善加算
- 日別勤怠：介護実労働／所定労働時間
- 月額給与の自動算出
- 時間外・深夜手当（基本給ベース）
- 経費精算／交通費／緊急時加算
- 標準報酬月額の管理
- 控除（保険料・源泉税）／手動調整フラグ
- 40歳以上は介護保険を自動加算

## 9. 請求実績インポート

### 9-1. 取込ページ（BillingImportPage）

- **CSV（Shift-JIS）／PDF** のドラッグ＆ドロップ取込
- 取込前プレビュー（先頭20件）／スキップ理由表示
- 重複排除＋バッチID自動採番
- ヘルパー別／利用者別フィルタ
- バッチごとの一括削除（履歴も削除）

- 曜日別色分け（日＝赤、土＝青）
- 集計：総件数／稼働日数／合計時間／ヘルパー数／利用者数／取込回数
- サービスコードのバッジ色分け（身体＝赤、家事＝緑、通院＝青など）

## 10. ケア日誌

**用途**：ヘルパーが訪問終了後に提出する日誌を管理者が確認・検収。

**できること**
- 日／週／月単位でフィルタリング
- ヘルパー別／利用者別フィルタ
- 記入状況の可視化（記入済み＝緑、未記入＝オレンジ警告）
- **出発・到着時刻の自動記録**確認
- 予定時間と実績時間の差分（±分、10分超過で橙色警告）
- 日誌詳細モーダル（ケア内容／体調／特記事項／今後の動き）
- 体調判定（良好／やや不良／不良）
- 古い日誌の削除

## 12. 書類スケジュール管理

### 12-2. タイムラインビュー（DocumentTimelineView）

- 利用者×書類種別の二次元タイムライン
- 月区切り線・本日ラベル
- 計画書作成日（白丸）と有効期間（半透明バー）の重ね描き
- モニタリング実施済（▲）／予定（△）の区別

## 13. 個人シフト表（ヘルパー向け）

**用途**：ヘルパーが自分のトークン付きURLからスマホで自分のシフトを確認。

**できること**
- 当月／前月／翌月切替
- 週ごとの5行レイアウト（月〜日固定7列）
- 時刻／利用者／サービス種別／時間数／地名表示
- キャンセル状態の赤色表示（時間残す／時間消すで区別）
- 休み希望（ピンク背景で日全体）
- 時間帯指定の休み希望表示
- 最終更新時刻表示
- **PWA対応**（ホーム画面追加ガイド・動的manifest）
- リアルタイム更新インジケーター

**アクセス方法**：管理画面でヘルパーごとの個人URLを生成 → コピーしてヘルパーに送付

## 14. 権限管理・ログイン

### 14-1. ログイン（Login）

- **Googleアカウント（OAuth 2.0）でのログイン**
- ホワイトリスト方式（事前登録アドレスのみ許可）
- 認証エラー時の詳細メッセージ表示
- メール／パスワード招待は未実装

### 14-2. 権限管理（PermissionManager、admin専用）

- ヘルパー一覧（削除済み除外）
- メールアドレス確認
- ドロップダウンで「管理者」「スタッフ」を切替
- 特別アカウント（info@alhena.co.jp）はロック
- 管理者人数のカウント表示

**権限による違い**
| 機能 | admin | staff |
|------|-------|-------|

## 15. メンテナンス機能

### 15-1. ケア内容削除ツール（CareContentDeleter）

- 日付単位／月単位でシフトを物理削除
- 削除前の件数確認
- **二重確認ダイアログ**（取り消し不可）
- 削除履歴ログ保存

### 15-2. 内部バックアップ（admin）

- 「☁️ 内部バックアップ」ボタン
- ヘルパー・シフト全データを Supabase 内部バックアップへ保存
- 災害復旧用途

### 15-3. 自動バックグラウンドCron

- **毎日深夜2時**：かんたん介護から利用者データを自動同期
- Netlify Functions + GitHub Actions Cron で稼働
