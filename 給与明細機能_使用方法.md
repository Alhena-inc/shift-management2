# 給与明細機能 使用方法

## 概要

シフト管理ソフトに給与明細の表示・印刷・PDF保存機能を追加しました。
Zapierを使わず、ソフト内で完結する仕様です。

---

## アクセス方法

### ローカル環境
```
http://localhost:5173/
```

### ネットワーク
```
http://192.168.10.111:5173/
```

---

## 使用手順

### 1. 給与計算画面を開く

1. メイン画面で対象の**年**と**月**を選択
2. 画面上部の「💰 給与計算」ボタンをクリック

### 2. 給与明細表示ボタンをクリック

給与計算画面の右上にある「**給与明細表示**」ボタン（青色）をクリック

### 3. ヘルパーを選択

給与明細表示モーダルが開きます。
- 表示したいヘルパーをクリックして選択
- 各ヘルパーの給与形態（時給/固定給）が表示されます

### 4. 給与明細を確認

選択したヘルパーの給与明細が表示されます。

**表示内容:**
- ヘッダー（会社情報、対象月）
- 基本情報（氏名、雇用形態、単価など）
- 勤怠項目（稼働日数）
- 稼働時間（通常、深夜、同行、事務・営業）
- 支給項目（各種報酬、経費、交通費、合計）
- 月勤怠表（日次の稼働時間）

### 5. 印刷またはPDF保存

**印刷する場合:**
1. 「🖨️ 印刷」ボタンをクリック
2. ブラウザの印刷ダイアログが開きます
3. プリンターを選択して印刷

**PDF保存する場合:**
1. 「📥 PDF保存」ボタンをクリック
2. ブラウザの印刷ダイアログが開きます
3. 送信先で「PDFに保存」を選択
4. 保存場所を指定して保存

## 単価設定

### 基本単価
- **基本単価**: 1,200円/時
- **処遇改善加算**: 800円/時
- **合計時間単価**: 2,000円/時

### サービス別単価
| サービス種別 | 単価 |
|-------------|------|
| 通常稼働（身体、重度、家事、通院、行動、移動） | 2,000円/時 |
| 同行 | 2,000円/時 |
| 事務・営業 | 1,200円/時 |
| 深夜（通常・同行） | 通常単価 × 1.5倍 |

### 報酬計算式
```
通常稼働報酬 = 通常稼働時間 × 2,000円
同行稼働報酬 = 同行時間 × 2,000円
事務・営業報酬 = 事務・営業時間 × 1,200円
深夜稼働報酬 = 深夜稼働時間 × 2,000円 × 1.5
深夜同行報酬 = 深夜同行時間 × 2,000円 × 1.5
支給額合計 = 各報酬 + 経費 + 交通費
```

---

## 深夜時間の計算

- **深夜時間帯**: 22:00〜翌8:00
- **深夜割増**: 通常単価の1.5倍
- 自動計算により、シフトの開始・終了時刻から深夜時間を抽出

## サービス種別の分類

### 通常稼働
以下のサービスを通常稼働としてカウント:
- 身体（shintai）
- 重度（judo）
- 家事（kaji）
- 通院（tsuin）
- 行動（kodo_engo）
- 移動（ido）

### 同行稼働
- 同行（doko）

### 事務・営業
- 事務（jimu）
- 営業（eigyo）

---

## 稼働日数のカウント方法

### 通常稼働日数
通常サービス（身体、重度、家事、通院、行動、移動）を1件以上行った日をカウント

### 同行稼働日数
同行サービスを1件以上行った日をカウント

### 合計稼働日数
通常稼働日と同行稼働日の重複を除いた合計日数

---

## 印刷・PDF保存のポイント

### 印刷設定の推奨
- **用紙サイズ**: A4
- **余白**: 標準（15mm）
- **背景のグラフィック**: ON（色付きセルを印刷するため）

### PDFファイル名の推奨
```
給与明細_ヘルパー名_YYYY年MM月.pdf
例: 給与明細_山田太郎_2024年12月.pdf
```

## トラブルシューティング

### 「給与明細表示」ボタンが表示されない
→ 給与計算画面を開いているか確認してください

### 給与データが正しく計算されない
→ シフトデータが正しく入力されているか確認してください
→ キャンセル状態（remove_time）のシフトは計算に含まれません

### 印刷時に色が出ない
→ ブラウザの印刷設定で「背景のグラフィック」をONにしてください

### PDFが保存できない
→ ブラウザの印刷ダイアログで「送信先」を「PDFに保存」に変更してください
